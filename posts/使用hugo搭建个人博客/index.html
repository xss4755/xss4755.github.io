使用hugo搭建github个人博客

龙小羽
 2021-02-04  约 1400 字   预计阅读 3 分钟 
目录

使用hugo静态网站生成器，选择LoveIt主题，将博客搭建搭建到github上。

1、本地安装hugo

本机系统macOS11.1，需要提前安装homebrew

hugo中文安装文章https://www.gohugo.org/

brew install hugo

2、本地创建博客

hugo new site myblog  #myblog目录就是创建的博客目录，创建在当前目录下

目录结构如下：

myblog/
|-- archetypes
|   -- default.md
|-- config.toml         #博客的配置文件
|-- content             #博客文章存放目录（markdown文件）
|-- data                    
|-- layouts
|-- static              #静态文件/图片/CSS/JS文件
|-- themes              #博客主题模板存放目录

3、配置博客主题

hugo中文文档的主题列表https://www.gohugo.org/theme/

我自己使用了LoveIt主题https://github.com/dillonzq/LoveIt/

cd myblog
git clone [https://github.com/dillonzq/LoveIt](https://github.com/dillonzq/LoveIt/blob/master/README.zh-cn.md) themes/LoveIt
vim config.toml  #将下面的配置复制到文件中，自行修改个人的资料

主题配置文件：

# 修改为博客的地址，这里使用github pages的地址
baseURL = "your_github_name.github.io"
# [en, zh-cn, fr, ...] 设置默认的语言
defaultContentLanguage = "zh-cn"
# 网站语言, 仅在这里 CN 大写
languageCode = "zh-CN"
# 是否包括中日韩文字
hasCJKLanguage = true
# 网站标题
title = "我的全新 Hugo 网站"

# 更改使用 Hugo 构建网站时使用的默认主题
theme = "LoveIt"

[params]
  # LoveIt 主题版本
  version = "0.2.X"
  # 网站描述
  description = "这是我的全新 Hugo 网站"
  # 网站关键词
  keywords = ["Theme", "Hugo"]
  # 网站默认主题样式 ("light", "dark", "auto")
  defaultTheme = "auto"

[menu]
  [[menu.main]]
    identifier = "posts"
    # 你可以在名称 (允许 HTML 格式) 之前添加其他信息, 例如图标
    pre = ""
    # 你可以在名称 (允许 HTML 格式) 之后添加其他信息, 例如图标
    post = ""
    name = "文章"
    url = "/posts/"
    # 当你将鼠标悬停在此菜单链接上时, 将显示的标题
    title = ""
    weight = 1
  [[menu.main]]
    identifier = "tags"
    pre = ""
    post = ""
    name = "标签"
    url = "/tags/"
    title = ""
    weight = 2
  [[menu.main]]
    identifier = "categories"
    pre = ""
    post = ""
    name = "分类"
    url = "/categories/"
    title = ""
    weight = 3

# Hugo 解析文档的配置
[markup]
  # 语法高亮设置 (https://gohugo.io/content-management/syntax-highlighting)
  [markup.highlight]
    # false 是必要的设置 (https://github.com/dillonzq/LoveIt/issues/158)
    noClasses = false

4、创建第一篇文章

创建第一篇文章的方法：

hugo new posts/first_post.md

vim content/posts/first_post.md

# 下方为文件内容
---    # 前置内容
title: "First_post"
date: 2021-02-04T13:36:55+08:00
draft: true         # 此处修改为false
---    # 下方是文章正文
### Hello World!

默认情况下, 所有文章和页面均作为草稿创建. 如果想要渲染这些页面, 请从元数据中删除属性 draft: true, 设置属性 draft: false 或者为 hugo 命令添加 -D/–buildDrafts 参数.

5、本地启动网站

hugo serve

访问http://localhost:1313查看我们的博客。

6、配置github

在github创建一个新的repository

”xxx.github.io“为项目名，其中xxx需要和你的github账户名相同，如下图所示。

7、部署hugo到github

hugo         # 生成静态文件public目录，其中包含网站的全部静态内容和资源。
cd public

# 先设置一下用户名和邮箱
git config --global user.name "龙小羽"
git config --global user.email "xss4755@gmail.com"

# 创建本地仓库
git init     # 在本地创建一个git仓库
git add .    # 将项目添加到缓存区
git commit -m "第一次提交"  # 将项目提交到Git仓库
git branch -M main

# 本地创建SSH key
ssh-keygen -t rsa -C "email@example.com" 
cd ~/.ssh    # 生成的ssh key在这里

github右上角头像点进去，找到设置

在”SSH and GPG keys“中，新建一个SSH keys，将之前~/.ssh目录下的id_rsa.pub内容复制进去

cd ~/xxxxxx/myblog/public  # 切回到生成的静态文件目录下

# 将本地仓库和远程仓库关联，后面跟的地址是刚刚创建的博客项目地址
git remote add origin https://github.com/xss4755/xss4755.github.io.git

git push -u origin main # 将本地仓库的东西推送到远程仓库，第一次需要加-u参数，后续不再需要。

push完成之后，我们访问自己的仓库名那个地址就可以访问我们的博客了~ https://xss4755.github.io/
