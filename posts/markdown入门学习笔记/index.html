很久之前也使用过markdown来码字，但中间由于种种原因(ps:就是懒，啥也别说了)，使用一段时间就放弃了。这次正好有时间，整理一下常用markdown的笔记。

同时因为markdown编辑器的不同，显示的效果也会有不同，大体还是一致的，不用太过纠结。

1. 简单使用

1.1 标题

标题有两种写法，我一般使用#来进行标记。

第一种：

’#‘+’空格‘+‘文字’标记1-6级标题

第二种：

使用=和-只能标记一级和二级标题(-和=只要超过两个就可以)

1.2 段落换行

markdown中换行新起一个段落需要先打两个空格再回车换行。有的编辑器可能直接回车就可以换行，但是为了兼容性还是打完两个空格在换行为上策。

1.3 字体

可以使用斜体、粗体，也可以同时加粗倾斜。可以使用*或者_，我一般选择*.(使用*不用切换输入法，哈哈~)

1.4 分割线、水平线

在一行内使用三个以上的星号、减号来添加一个分割线、水平线，这一行中不能再输入其他内容，可以加空格。

1.5 删除线

在文字两端都加两个波浪线，就可以在需要保留文字的同时添加删除线。

2. 写代码、写文章

2.1 列表

列表包括有序和无序两种。

无序列表

使用星号、加号、减号进行标记，标记后面要加空格，在写文字。

有序列表

使用‘数字’ + ‘.’ + ‘文字’

列表嵌套

在子列表里面加四个空格即可。

2.2 区块引用

想对上面的内容做一些说明或者引用其他人的内容时使用，使用'>'，可以多级引用。

引用
> 佛说：  
> 要有光

* 我是谁
    > 我来自何方
    > 像一颗尘土
        >啊哈？

引用

佛说：
要有光

  • 我是谁

    我来自何方 像一颗尘土

    啊哈？

2.3 代码

**单行代码 **

使用反引号`引起来。

一行代码

**代码块 **

前后都使用三个反引号引起来。

# 这是一个代码块
print('1')

2.4 超链接

两种方式

[链接名称](链接地址)
<链接地址>

例如：
超链接在这里，[百度](https://www.baidu.com)
<https://www.baidu.com>

超链接在这里，百度

https://www.baidu.com

2.5 图片

![这是一个图片](https://miao.xss4755.top/blog/2020-10-15-053038.png "freebuf")

[]是图片的属性，可以不写。
()是图片地址，可以在里面的后面加""写图片名字。
可以单独放，也可以放在正文中。

2.6 表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

还可以设置对齐方式：

  • -: 左对齐
  • :- 右对齐
  • :-: 居中对齐
|  表头   | 表头  |
|  :-:  | :-:  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |
表头 表头
单元格 单元格
单元格 单元格

*2.7 脚注

这里有[^小老虎]

[^小老虎]:一种小动物
